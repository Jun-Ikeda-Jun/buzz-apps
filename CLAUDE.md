# buzz-apps

## 概要
「溺愛 × 画像加工」をテーマにしたバズ狙いWebアプリ7連発。
インスピレーションキャットの構造（ローカル処理・コスト0円・SNS拡散ループ）を踏襲し、全部出して世の中に評価してもらう。

## 技術スタック
- HTML / CSS / JavaScript（フレームワークなし、軽量最優先）
- 背景除去: onnxruntime-web + 軽量セグメンテーションモデル（WASM実行）
- Canvas API: 画像合成・テキスト描画
- デプロイ: GitHub Pages（無料）
- ビルドツール: なし（静的ファイルのみ）

## アプリ一覧
1. ペット写真 → 雑誌風表紙
2. 推し写真 → 祭壇/痛バ風画像
3. 赤ちゃん写真 → 月齢入り成長カード
4. ぬい写真 → 世界の名所に合成
5. ペット写真 → LINEスタンプ風カット
6. 料理写真 → 雑誌レイアウト化
7. 車写真 → 映画ポスター風

## ディレクトリ構成
```
buzz-apps/
  shared/           # 共通ライブラリ（背景除去、Canvas操作）
  apps/
    pet-magazine/   # 1. ペット雑誌表紙
    oshi-altar/     # 2. 推し祭壇
    baby-card/      # 3. 赤ちゃん月齢カード
    nui-travel/     # 4. ぬい旅行
    pet-stamp/      # 5. ペットスタンプ
    food-magazine/  # 6. 料理雑誌
    car-poster/     # 7. 車映画ポスター
  index.html        # ポートフォリオ（7つへのリンク）
```

## 開発ルール
- 1アプリ = 1ディレクトリ、独立して動作する
- 全処理ブラウザローカル（サーバーなし）
- 画像はサーバーに送信しない
- 共通コードは shared/ に置くが、各アプリは単体でも動く
- テンプレ構造を1個目で固め、2個目以降は量産
